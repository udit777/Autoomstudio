### 8.2 Typography

### 8.3 CORE ANIMATION REQUIREMENTS

#### 8.3.1 Hero Automation Workflow Animation

**Animation Characteristics:**
- **Theme:** Automated workflow with data flowing through systems
- **Path:** Circular or interconnected node pattern
- **Speed:** 8-10 second loop (continuous operation feel)
- **Elements:**
  - Connected nodes (process steps)
  - Data flows between nodes
  - Pulses indicating automation actions
  - Status indicators (green for active)
  - Process efficiency visualizations

**Trigger:** Auto-play on page load with smooth fade-in
**Interaction:** Pause on hover, resume on unhover
**Mobile:** Simplified animation with reduced node count

---

#### 8.3.2 Scroll-Triggered Animations

**Elements with Scroll Triggers:**
- Service cards fade in on scroll
- Counter animations (0 → final number)
- Text reveals with staggered timing
- Feature lists slide in from left
- Images scale up on viewport entry
- Testimonial cards fade in
- Stats grow from 0%
- Service comparison table reveals

**Animation Timing:**
- Entrance: 0.6-0.8 seconds
- Stagger: 0.1-0.15 seconds between elements
- Easing: ease-out for smooth, natural feel

---

#### 8.3.3 Interactive Hover Effects

**Service Cards:**
- Background color shift to accent color (subtle)
- Scale: 1 → 1.05
- Shadow increase (elevation effect)
- Icon animation (scale, rotate)
- Text color change
- "Learn More" link appears/highlights

**Navigation Items:**
- Underline animation (width 0 → 100%)
- Text color transition
- Background highlight (subtle)
- Dropdown arrows rotate smoothly

**Buttons:**
- Background color transition
- Text color change
- Icon animation (arrow slides right)
- Ripple effect on click (optional)
- Shadow increase on hover

**Images & Media:**
- Brightness increase (+10%)
- Slight zoom (1 → 1.08)
- Overlay color fade
- Lightbox trigger on click

---

#### 8.3.4 Data Visualization Animations

**Counters:**
- Count from 0 to final number
- Duration: 2-3 seconds
- Animation: ease-out
- Can display: Projects, Clients, Team members, Success rate

**Charts & Graphs:**
- Bar charts grow from 0
- Line charts draw themselves
- Pie charts rotate into view
- Legend items fade in sequentially

**Process Flows:**
- Step-by-step process animation
- Numbers count down
- Arrows animate between steps
- Timeline dots fill in progressively

---

#### 8.3.5 Page Transitions & Loading

**Page Fade Transitions:**
- Fade out current page (0.2 seconds)

**Loading States:**

## 9. TECHNICAL SPECIFICATIONS

### 9.1 Antigravity Implementation

**Core Technologies:**
- **Framework:** Antigravity (modern web framework)
- **Animation Library:** Built-in Antigravity animations + Framer Motion or similar
- **Data Visualization:** D3.js or Chart.js for complex charts
- **3D Elements:** Three.js for any 3D visualizations (optional)
- **State Management:** Antigravity state system
- **CMS:** Headless CMS for blog and case studies (Contentful, Sanity, or similar)

### 9.2 Performance Targets

**Core Web Vitals:**
- **LCP (Largest Contentful Paint):** < 2.5 seconds
- **FID (First Input Delay):** < 100ms
- **CLS (Cumulative Layout Shift):** < 0.1
- **Page Load Time:** < 2 seconds (target)

**Performance Optimization:**
- Image optimization and lazy loading
- Code splitting by page
- CSS-in-JS with critical CSS extraction
- Font loading optimization
- Caching strategies

### 9.3 SEO Optimization

**On-Page SEO:**
- Meta tags (title, description, keywords) for each page
- Schema.org structured data markup:
  - Organization schema
  - Service schema
  - LocalBusiness schema
  - BreadcrumbList schema
  - FAQPage schema
- Open Graph tags for social sharing
- Twitter Card tags
- Mobile-first indexing compliance

**Technical SEO:**
- XML sitemap with all pages
- Robots.txt with proper directives
- 301 redirects for URL changes
- Canonical tags
- Structured data validation

**Content SEO:**
- Keyword optimization for service pages
- Internal linking strategy
- Header hierarchy (H1 → H6)
- Image alt text and optimization
- Long-form content for ranking
